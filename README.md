### 📌 *Sobre o Projeto*

 *Este projeto é um sistema simples de conta bancária desenvolvido em Java, utilizando os princípios de Programação Orientada a Objetos (POO).*
 *A aplicação permite ao usuário interagir via terminal com um menu para realizar operações bancárias básicas.*

---
### ⚙️ Funcionalidades

* Criar conta bancária
* Depositar valores
* Realizar saques
* Consultar saldo
* Validação de valores inválidos
* Menu interativo no terminal

---
### 🧠 *Conceitos Aplicados*
* Programação Orientada a Objetos (POO)
* Encapsulamento
* Classes e Métodos
* Estrutura de decisão (switch)
* Estrutura de repetição (do-while)
* Entrada de dados com Scanner
